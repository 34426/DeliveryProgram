## 功能说明

系统角色 ：用户(小程序端) 配送员 商家 管理员

用户、配送员、商家、管理员

用户(小程序端)：注册、登录、菜品浏览、菜品检索、菜品收藏/评论/加入购物车/立即购买、购物车结算、地址维护、用户充值、订单管理、个人信息

配送员：订单查询、接单、订单配送、送达登记

商家：菜品管理、订单查询、订单发货、订单接单查询、订单配送查询、订单送达查询、公告查询

管理员：用户管理、配送员管理、商家管理、菜品管理、订单管理、公告管理。

技术说明

环境：JoK、MySQL、Node

框架 ：SpringBoot 、Vue 、Element-UI

小程序 ：原生开发
